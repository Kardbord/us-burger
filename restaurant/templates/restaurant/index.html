About

The following steps outline how to navigate the website and view the changes both from the app and on the admin page.

1. Perform a git pull to ensure that you have the most recent changes. We're no longer using the same `db.sqlite` file, so this shouldn't be as big of an issue as it has been in the past.

2. Run the following commands from the terminal within the project directory: * `$ python manage.py makemigrations restaurant` * `$ python manage.py migrate` * `$ python manage.py runserver`

3. Navigate to [localhost:8000/init](localhost:8000/init) to run Tanner K.'s populate method to wipe any local MenuItem entries and populate everything with our 'defaults'.
